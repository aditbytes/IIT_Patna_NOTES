# 📚 IIT Notes Repository

A structured collection of notes, resources, and references for IIT courses.  
Built for fast revision, clear concepts, and efficient exam preparation.

---

## 🚀 Purpose

IIT courses move fast, and unorganized notes waste time. This repository aims to:

- Organize notes semester-wise and subject-wise  
- Keep content concise and exam-focused  
- Provide quick revision material

## 📖 What's Included

- **Class Notes** – Clean and structured lecture notes  
- **Concept Summaries** – Simplified key ideas  
- **Cheat Sheets** – Quick revision material  
- **Assignments / Practice Problems** (if available)  
- **Reference Resources**  

---

## 🧩 How to Use

Use this repo properly:

1. Review notes after lectures  
2. Identify weak areas  
3. Practice related problems  
4. Revise using cheat sheets before exams  

Passive reading won’t help—practice is essential.

---

## 🛠️ Contribution Guidelines

- Keep notes concise and structured  
- Avoid uploading raw, unorganized files  
- Use clear headings and formatting  
- Follow the existing folder structure  

**Steps:**
1. Fork the repository  
2. Make your changes  
3. Submit a Pull Request
