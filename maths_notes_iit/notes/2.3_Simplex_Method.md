## What is the Simplex Method? (Layman's Explanation)

The graphical method works only for 2 variables. For real-world problems with hundreds of variables, we need something better — the **Simplex Method**.

**Idea:** Instead of checking ALL corners (which could be millions), the Simplex Method **starts at one corner** and **moves to an adjacent better corner** step by step, until it reaches the **optimal corner**. It's like climbing a hill by always taking the steepest upward step.

---

## Key Terminology

### Basic and Non-Basic Variables

For a system $AX = B$ with $m$ equations and $n$ variables ($n > m$):

- **Basic variables:** $m$ variables (one for each equation). Their values are determined by the equations.
- **Non-basic variables:** $n - m$ remaining variables, set to **zero**.

### Basic Feasible Solution (BFS)
A **basic solution** where all basic variables are **≥ 0** (non-negative).

### Degenerate BFS
A BFS where one or more basic variables equals **zero**.

### Optimal BFS
A BFS that gives the **maximum (or minimum)** value of the objective function.

## Setting Up for Simplex

### Step 1: Convert to Standard Form

All constraints must be **equalities** with $b_i \geq 0$.

**For $\leq$ constraints:** Add **slack variable** $s_i \geq 0$

$$
2x_1 + 3x_2 \leq 12 \implies 2x_1 + 3x_2 + s_1 = 12
$$

**For $\geq$ constraints:** Subtract **surplus variable** $s_i \geq 0$

$$
x_1 + 4x_2 \geq 8 \implies x_1 + 4x_2 - s_1 = 8
$$

(For $\geq$ constraints, we also need artificial variables — see Topic 2.4)

### Step 2: Write in Tabular Form

The **Simplex Tableau** is a table that tracks all the information at each step.

---

## The Simplex Tableau

For **Maximize** $Z = c_1x_1 + c_2x_2 + \cdots + c_nx_n$

Subject to $AX = B$, $X \geq 0$

| $c_B$ | Basis | $b$ | $x_1$ | $x_2$ | $\cdots$ | $s_1$ | $s_2$ | $\cdots$ | Ratio |
|-------|-------|-----|--------|--------|----------|--------|--------|----------|-------|
| $0$ | $s_1$ | $b_1$ | $a_{11}$ | $a_{12}$ | | $1$ | $0$ | | |
| $0$ | $s_2$ | $b_2$ | $a_{21}$ | $a_{22}$ | | $0$ | $1$ | | |
| | | | $Z_j - c_j$ | $Z_j - c_j$ | | | | | |

Where:
- **$c_B$** = coefficients of basic variables in the objective function
- **Basis** = current basic variables
- **$b$** = right-hand side values (current solution)
- **$Z_j - c_j$** = optimality indicators (bottom row)

---

## Simplex Algorithm (Step by Step)

### Step 1: Check Optimality
Look at the bottom row ($Z_j - c_j$ values, also called **reduced costs**).

**For Maximization:**
- If all $Z_j - c_j \geq 0$ → **Optimal solution found!** STOP.
- If any $Z_j - c_j < 0$ → Not optimal. Go to Step 2.

**For Minimization:**
- If all $Z_j - c_j \leq 0$ → **Optimal solution found!** STOP.
- If any $Z_j - c_j > 0$ → Not optimal. Go to Step 2.

### Step 2: Choose Entering Variable (Pivot Column)
Select the column with the **most negative** $Z_j - c_j$ value (for maximization).

This variable will **enter** the basis.

### Step 3: Choose Leaving Variable (Pivot Row)
For each row where the pivot column entry is **positive**, compute the **ratio**:

$$
\text{Ratio} = \frac{b_i}{\text{pivot column entry in row } i}
$$

Select the row with the **smallest non-negative ratio** (minimum ratio rule).

This variable will **leave** the basis.

**If no positive entry exists in the pivot column → Solution is UNBOUNDED.**

### Step 4: Pivot Operation
The element at the intersection of pivot row and pivot column is the **pivot element**.

1. **Divide the pivot row** by the pivot element (make it 1)
2. **Eliminate** the pivot column entries in all other rows (make them 0) using row operations:
   - $R_i \to R_i - (\text{entry in pivot column}) \times (\text{new pivot row})$

### Step 5: Update Basis
Replace the leaving variable with the entering variable. Update $c_B$.

### Step 6: Go back to Step 1.

## Complete Example 1: Maximization

**Maximize:** $Z = 3x_1 + 5x_2$

**Subject to:**
$$
x_1 + 2x_2 \leq 12
$$
$$
2x_1 + x_2 \leq 12
$$
$$
x_1, x_2 \geq 0
$$

### Setup: Add slack variables

$x_1 + 2x_2 + s_1 = 12$

$2x_1 + x_2 + s_2 = 12$

Maximize $Z = 3x_1 + 5x_2 + 0s_1 + 0s_2$

### Initial Tableau

| $c_B$ | Basis | $b$ | $x_1$ (3) | $x_2$ (5) | $s_1$ (0) | $s_2$ (0) | Ratio |
|-------|-------|-----|-----------|-----------|-----------|-----------|-------|
| 0 | $s_1$ | 12 | 1 | **2** | 1 | 0 | 12/2 = 6 |
| 0 | $s_2$ | 12 | 2 | 1 | 0 | 1 | 12/1 = 12 |
| | $Z_j - c_j$ | 0 | **-3** | **-5** | 0 | 0 | |

**$Z_j - c_j$ row:** $Z_j = c_B \cdot \text{column}$. For $x_1$: $0(1) + 0(2) - 3 = -3$. For $x_2$: $0(2) + 0(1) - 5 = -5$.

**Step 1:** Not optimal (negative values exist).

**Step 2:** Most negative = $-5$ → $x_2$ enters (pivot column).

**Step 3:** Ratios: 12/2 = 6, 12/1 = 12. Minimum = 6 → $s_1$ leaves (Row 1).

**Pivot element:** 2 (Row 1, $x_2$ column).

### Step 4: Pivot

**New Row 1:** $R_1 / 2$: $[6, 1/2, 1, 1/2, 0]$

**New Row 2:** $R_2 - 1 \times \text{New } R_1$: $[12 - 6, 2 - 1/2, 1 - 1, 0 - 1/2, 1] = [6, 3/2, 0, -1/2, 1]$

### Tableau 2

| $c_B$ | Basis | $b$ | $x_1$ (3) | $x_2$ (5) | $s_1$ (0) | $s_2$ (0) | Ratio |
|-------|-------|-----|-----------|-----------|-----------|-----------|-------|
| 5 | $x_2$ | 6 | 1/2 | 1 | 1/2 | 0 | 6/(1/2) = 12 |
| 0 | $s_2$ | 6 | **3/2** | 0 | -1/2 | 1 | 6/(3/2) = 4 |
| | $Z_j - c_j$ | 30 | **-1/2** | 0 | 5/2 | 0 | |

$Z_j - c_j$ for $x_1$: $5(1/2) + 0(3/2) - 3 = 5/2 - 3 = -1/2$

**Step 1:** Not optimal ($-1/2 < 0$).

**Step 2:** $x_1$ enters.

**Step 3:** Min ratio = 4 → $s_2$ leaves. Pivot element = $3/2$.

### Step 4: Pivot

**New Row 2:** $R_2 / (3/2)$: $[4, 1, 0, -1/3, 2/3]$

**New Row 1:** $R_1 - (1/2) \times \text{New } R_2$: $[6-2, 1/2-1/2, 1, 1/2+1/6, -1/3] = [4, 0, 1, 2/3, -1/3]$

### Tableau 3 (Final)

| $c_B$ | Basis | $b$ | $x_1$ (3) | $x_2$ (5) | $s_1$ (0) | $s_2$ (0) |
|-------|-------|-----|-----------|-----------|-----------|-----------|
| 5 | $x_2$ | 4 | 0 | 1 | 2/3 | -1/3 |
| 3 | $x_1$ | 4 | 1 | 0 | -1/3 | 2/3 |
| | $Z_j - c_j$ | 32 | 0 | 0 | 7/3 | 1/3 |

**All $Z_j - c_j \geq 0$ → OPTIMAL!**

**Optimal Solution:** $x_1 = 4$, $x_2 = 4$, **$Z_{\max} = 32$**

(Same answer as graphical method! ✓)

## Complete Example 2: Another Maximization

**Maximize:** $Z = 5x_1 + 4x_2$

**Subject to:**
$$
6x_1 + 4x_2 \leq 24
$$
$$
x_1 + 2x_2 \leq 6
$$
$$
x_1, x_2 \geq 0
$$

### Standard form:
$6x_1 + 4x_2 + s_1 = 24$, $x_1 + 2x_2 + s_2 = 6$

### Initial Tableau

| $c_B$ | Basis | $b$ | $x_1$ (5) | $x_2$ (4) | $s_1$ (0) | $s_2$ (0) | Ratio |
|-------|-------|-----|-----------|-----------|-----------|-----------|-------|
| 0 | $s_1$ | 24 | 6 | 4 | 1 | 0 | 24/6 = 4 |
| 0 | $s_2$ | 6 | 1 | 2 | 0 | 1 | 6/1 = 6 |
| | $Z_j - c_j$ | 0 | -5 | -4 | 0 | 0 | |

$x_1$ enters (most negative = $-5$). Min ratio = 4 → $s_1$ leaves. Pivot = 6.

### Tableau 2

New $R_1 = R_1 / 6$: $[4, 1, 2/3, 1/6, 0]$

New $R_2 = R_2 - 1 \times$ New $R_1$: $[2, 0, 4/3, -1/6, 1]$

| $c_B$ | Basis | $b$ | $x_1$ | $x_2$ | $s_1$ | $s_2$ | Ratio |
|-------|-------|-----|-------|-------|-------|-------|-------|
| 5 | $x_1$ | 4 | 1 | 2/3 | 1/6 | 0 | 4/(2/3) = 6 |
| 0 | $s_2$ | 2 | 0 | 4/3 | -1/6 | 1 | 2/(4/3) = 3/2 |
| | $Z_j - c_j$ | 20 | 0 | -2/3 | 5/6 | 0 | |

Not optimal ($-2/3 < 0$). $x_2$ enters. Min ratio = 3/2 → $s_2$ leaves. Pivot = 4/3.

### Tableau 3 (Final)

New $R_2 = R_2 / (4/3)$: $[3/2, 0, 1, -1/8, 3/4]$

New $R_1 = R_1 - (2/3) \times$ New $R_2$: $[4 - 1, 1, 0, 1/6 + 1/12, -1/2] = [3, 1, 0, 1/4, -1/2]$

| $c_B$ | Basis | $b$ | $x_1$ | $x_2$ | $s_1$ | $s_2$ |
|-------|-------|-----|-------|-------|-------|-------|
| 5 | $x_1$ | 3 | 1 | 0 | 1/4 | -1/2 |
| 4 | $x_2$ | 3/2 | 0 | 1 | -1/8 | 3/4 |
| | $Z_j - c_j$ | 21 | 0 | 0 | 3/4 | 1/2 |

**All $Z_j - c_j \geq 0$ → OPTIMAL!**

**Optimal Solution:** $x_1 = 3$, $x_2 = 3/2$, **$Z_{\max} = 21$** ✓

## Important Special Cases

### 1. Unbounded Solution
If in Step 3, **all entries in the pivot column are ≤ 0**, then no variable can leave the basis.

→ The solution is **unbounded** (Z can increase without limit).

### 2. Degenerate Solution
If in Step 3, there's a **tie in the minimum ratio** (two rows have the same ratio), degeneracy occurs.

→ One of the basic variables will become **zero** in the next iteration.
→ May cause **cycling** (rare in practice). Use **Bland's Rule** to avoid it.

### 3. Multiple Optimal Solutions
If in the final tableau, a **non-basic variable** has $Z_j - c_j = 0$, then there are **multiple optimal solutions**.

→ Bringing that variable into the basis gives another optimal BFS with the same $Z$ value.

### 4. Infeasible Problem
If using Big-M or Two-Phase method, an **artificial variable remains in the basis** with a positive value in the final tableau.

→ The problem is **infeasible**.

## Computing $Z_j - c_j$ (Detailed)

$$
Z_j = \sum_{i=1}^{m} c_{B_i} \cdot a_{ij}
$$

Where $c_{B_i}$ is the objective coefficient of the $i$-th basic variable, and $a_{ij}$ is the entry in row $i$, column $j$ of the tableau.

$$
\text{Reduced cost} = Z_j - c_j
$$

The current value of $Z$:
$$
Z = \sum_{i=1}^{m} c_{B_i} \cdot b_i
$$

---

## Minimization Using Simplex

**Two approaches:**

### Approach 1: Convert to Maximization
Minimize $Z$ = Maximize $Z' = -Z$

Solve the max problem, then negate the answer.

### Approach 2: Modify Optimality Condition
For **minimization**, the optimal condition is: all $Z_j - c_j \leq 0$.

Choose the **most positive** $Z_j - c_j$ as the entering variable.

**Solution outline:**
Initial BFS: $s_1 = 8, s_2 = 12$, $Z = 0$.

$x_1$ enters → $s_2$ leaves → After pivot: $x_1 = 2, x_2 = 3, Z = 13$.
All $Z_j - c_j \geq 0$ → **Optimal: $Z = 13$** at $(2, 3)$.

## Key Formulas to Remember

| Formula | Description |
|---------|-------------|
| $Z_j = \sum c_{B_i} \cdot a_{ij}$ | Computing indicator row |
| Min ratio = $\min(b_i / a_{ij})$ for $a_{ij} > 0$ | Leaving variable selection |
| Most negative $Z_j - c_j$ → entering | Entering variable (Max) |
| All $Z_j - c_j \geq 0$ → Optimal (Max) | Optimality condition |
| All $Z_j - c_j \leq 0$ → Optimal (Min) | Optimality condition |
| All pivot column entries ≤ 0 → Unbounded | Unbounded detection |
