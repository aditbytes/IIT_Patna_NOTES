## Why Do We Need Artificial Variables? (Layman's Explanation)

The basic Simplex method needs an **initial basic feasible solution** to start. When constraints are $\leq$ type, slack variables naturally form an identity matrix and give us an easy starting point.

But what about $\geq$ or $=$ constraints? Adding a surplus variable gives a $-1$ coefficient, which can't be part of an identity matrix.

**Solution:** We add **artificial variables** to create an identity matrix, then use clever techniques to **drive them out** of the solution.

---

## When Are Artificial Variables Needed?

| Constraint Type | What to Add | Artificial Needed? |
|----------------|-------------|-------------------|
| $\leq$ | Slack variable ($+s_i$) | âťŚ No |
| $\geq$ | Surplus variable ($-s_i$) + Artificial ($+a_i$) | âś… Yes |
| $=$ | Artificial variable ($+a_i$) only | âś… Yes |

## Method 1: Big-M Method (Penalty Method)

### Core Idea
Add artificial variables to the objective function with a very **large penalty** $M$ (where $M \to \infty$):

- **Maximization:** Subtract $Ma_i$ â†’ $Z = c^TX - Ma_1 - Ma_2 - \cdots$
- **Minimization:** Add $Ma_i$ â†’ $Z = c^TX + Ma_1 + Ma_2 + \cdots$

The penalty ensures that in the optimal solution, **artificial variables become zero** (they're too "expensive" to keep).

### Step-by-Step Process

1. Convert all constraints to equalities (add slack/surplus/artificial as needed)
2. Add $-M \cdot a_i$ terms (for max) to the objective function
3. Set up initial simplex tableau with artificial variables in the basis
4. Apply the simplex method as usual
5. If all artificial variables leave the basis â†’ **Feasible solution found**
6. If any artificial variable remains in basis with positive value â†’ **Problem is infeasible**

---

### Big-M Example: Minimization

**Minimize:** $Z = 4x_1 + x_2$

**Subject to:**
$$
3x_1 + x_2 = 3
$$
$$
4x_1 + 3x_2 \geq 6
$$
$$
x_1 + 2x_2 \leq 4
$$
$$
x_1, x_2 \geq 0
$$

### Step 1: Standard form with artificials

- Eq 1 ($=$): Add artificial $a_1$ â†’ $3x_1 + x_2 + a_1 = 3$
- Eq 2 ($\geq$): Subtract surplus, add artificial â†’ $4x_1 + 3x_2 - s_1 + a_2 = 6$
- Eq 3 ($\leq$): Add slack â†’ $x_1 + 2x_2 + s_2 = 4$

### Step 2: Objective function

**Minimize:** $Z = 4x_1 + x_2 + 0s_1 + 0s_2 + Ma_1 + Ma_2$

### Step 3: Initial BFS

Basic variables: $a_1 = 3$, $a_2 = 6$, $s_2 = 4$

### Initial Tableau (for minimization)

| $c_B$ | Basis | $b$ | $x_1$ (4) | $x_2$ (1) | $s_1$ (0) | $s_2$ (0) | $a_1$ (M) | $a_2$ (M) |
|-------|-------|-----|-----------|-----------|-----------|-----------|-----------|-----------|
| $M$ | $a_1$ | 3 | 3 | 1 | 0 | 0 | 1 | 0 |
| $M$ | $a_2$ | 6 | 4 | 3 | -1 | 0 | 0 | 1 |
| $0$ | $s_2$ | 4 | 1 | 2 | 0 | 1 | 0 | 0 |

**$Z_j - c_j$ row:**

For $x_1$: $M(3) + M(4) + 0(1) - 4 = 7M - 4$

For $x_2$: $M(1) + M(3) + 0(2) - 1 = 4M - 1$

For minimization, choose the **most positive** $Z_j - c_j$ to enter.

Most positive: $7M - 4$ â†’ $x_1$ enters.

Ratios: $3/3 = 1$, $6/4 = 1.5$, $4/1 = 4$ â†’ Min = 1 â†’ $a_1$ leaves. Pivot = 3.

Continue simplex iterations until all $Z_j - c_j \leq 0$.

**Final answer (after iterations):** $x_1 = 2/5$, $x_2 = 9/5$, $Z_{\min} = 25/5 = 5$.

### Big-M Example: Maximization

**Maximize:** $Z = 3x_1 + 2x_2$

**Subject to:**
$$
x_1 + x_2 \leq 4
$$
$$
x_1 + x_2 \geq 2
$$
$$
x_1, x_2 \geq 0
$$

Standard form:
- $x_1 + x_2 + s_1 = 4$
- $x_1 + x_2 - s_2 + a_1 = 2$

Maximize: $Z = 3x_1 + 2x_2 + 0s_1 + 0s_2 - Ma_1$

Initial BFS: $s_1 = 4$, $a_1 = 2$

$Z_j - c_j$ for $x_1$: $0 \cdot 1 + (-M) \cdot 1 - 3 = -M - 3$

$x_1$ enters (most negative). Min ratio: $s_1$ row: 4/1 = 4, $a_1$ row: 2/1 = 2. $a_1$ leaves.

After pivoting: $x_1 = 2$, $s_1 = 2$, $Z = 6$.

Check $Z_j - c_j$: all â‰Ą 0 â†’ **Optimal: $Z = 6$ at $(2, 0)$**.

## Method 2: Two-Phase Method

### Core Idea
Instead of using a penalty $M$, solve the problem in **two separate phases**.

### Phase I: Find a Basic Feasible Solution

**Objective:** Minimize $W = a_1 + a_2 + \cdots + a_k$ (sum of all artificial variables)

Subject to the same constraints (with artificial variables).

If $W_{\min} = 0$ â†’ A BFS exists. Go to Phase II.

If $W_{\min} > 0$ â†’ **Problem is infeasible.** STOP.

### Phase II: Optimize the Original Objective

Start from the BFS obtained in Phase I. Drop all artificial variable columns. Use the original objective function. Apply simplex method.

---

### Two-Phase Example

**Minimize:** $Z = x_1 - 2x_2 - 3x_3$

**Subject to:**
$$
-2x_1 + x_2 + 3x_3 = 2
$$
$$
2x_1 + 3x_2 + 4x_3 = 1
$$
$$
x_1, x_2, x_3 \geq 0
$$

### Phase I Setup

Both constraints are $=$ type, so add artificial variables $a_1$ and $a_2$:

$-2x_1 + x_2 + 3x_3 + a_1 = 2$

$2x_1 + 3x_2 + 4x_3 + a_2 = 1$

**Phase I Objective:** Minimize $W = a_1 + a_2$

### Phase I: Initial Tableau

| $c_B^W$ | Basis | $b$ | $x_1$ | $x_2$ | $x_3$ | $a_1$ | $a_2$ |
|---------|-------|-----|--------|--------|--------|--------|--------|
| 1 | $a_1$ | 2 | -2 | 1 | 3 | 1 | 0 |
| 1 | $a_2$ | 1 | 2 | 3 | 4 | 0 | 1 |

$W_j - c_j^W$: For $x_1$: $1(-2) + 1(2) - 0 = 0$; For $x_2$: $1(1) + 1(3) - 0 = 4$; For $x_3$: $1(3) + 1(4) - 0 = 7$.

For minimization of $W$: choose most positive â†’ $x_3$ enters (value 7).

Ratios: $2/3$ and $1/4$. Min = $1/4$ â†’ $a_2$ leaves. Pivot = 4.

Continue until $W = 0$ or determine infeasibility.

### Phase II

Once $W = 0$, drop $a_1$ and $a_2$ columns. Use the BFS from Phase I as the starting point.

Replace the objective row with the original $Z = x_1 - 2x_2 - 3x_3$, compute new $Z_j - c_j$ values, and continue simplex iterations.

## Comparison: Big-M vs Two-Phase

| Feature | Big-M Method | Two-Phase Method |
|---------|-------------|------------------|
| **Number of phases** | One | Two |
| **Penalty parameter** | Uses large $M$ | No $M$ needed |
| **Computational issue** | $M$ can cause numerical problems | More stable |
| **Infeasibility detection** | Artificial in final basis | $W_{\min} > 0$ in Phase I |
| **When to use** | Good for hand calculations | Better for computers |
| **Simplicity** | Simpler setup | Slightly more steps |
| **Professor preference** | Both taught equally | Both may appear in exam |

## Additional Important Concepts

### Basis Notation

A basis $B$ is an $m \times m$ non-singular submatrix of $A$.

The basic solution is: $X_B = B^{-1}b$

The non-basic variables: $X_N = 0$

The objective value: $Z = c_B^T B^{-1} b$

### Simplex Multipliers (Dual Variables)

$$
Y^T = c_B^T B^{-1}
$$

These are related to the **shadow prices** in duality theory.

### Reduced Cost Interpretation

$Z_j - c_j = c_B^T B^{-1} A_j - c_j$

- For basic variables: $Z_j - c_j = 0$ always
- For non-basic variables: tells how much $Z$ changes per unit increase

## Key Formulas to Remember

| Formula | Description |
|---------|-------------|
| $\leq$ â†’ add $+s_i$ | Slack variable |
| $\geq$ â†’ add $-s_i + a_i$ | Surplus + Artificial |
| $=$ â†’ add $+a_i$ | Artificial only |
| Max: $Z - Ma_i$ | Big-M penalty (maximization) |
| Min: $Z + Ma_i$ | Big-M penalty (minimization) |
| Phase I: Min $W = \sum a_i$ | Two-Phase first objective |
| $W_{\min} = 0$ â†’ feasible | Two-Phase feasibility check |
| $W_{\min} > 0$ â†’ infeasible | No feasible solution |
