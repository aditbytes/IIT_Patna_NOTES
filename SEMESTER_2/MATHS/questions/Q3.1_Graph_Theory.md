## 🟢 Easy (Q1–Q9)

**Q1.** Define a graph. What are vertices and edges?

**Solution:**
A **graph** $G=(V,E)$ consists of:
- $V$ = non-empty set of **vertices** (nodes)
- $E$ = set of **edges** (unordered pairs $\{u,v\}$ for undirected, ordered pairs $(u,v)$ for directed)

Two vertices connected by an edge are **adjacent**. An edge connects its two **endpoints**.

---

**Q2.** State the Handshaking Theorem.

**Solution:**
For any undirected graph $G=(V,E)$:
$$\sum_{v\in V}\deg(v) = 2|E|$$
The sum of all vertex degrees equals twice the number of edges. Therefore, the number of odd-degree vertices is always **even**.

---

**Q3.** A graph has 6 edges. The degrees are 1,1,2,2,2,3,3. Verify the Handshaking Theorem.

**Solution:**
Sum of degrees $= 1+1+2+2+2+3+3 = 14$.

But $2|E| = 2\times6 = 12 \neq 14$.

Hmm — the problem data is inconsistent. For a valid graph with 6 edges: $\sum\deg=12$. Example: degrees $1,1,2,2,2,2,2$ sum to $12$. Let's use: degrees $2,2,2,2,2,2$ for $K_4$ or similar.

**Correct example:** Graph with degrees $1,1,2,2,2,2$ → sum$=10\neq12$. Try $2,2,2,2,2,2$→sum$=12=2(6)$. ✓

**Q4.** Define: (a) simple graph, (b) multigraph, (c) complete graph $K_n$.

**Solution:**
- **(a) Simple graph:** No self-loops, no multiple edges between the same pair of vertices.
- **(b) Multigraph:** Multiple edges (parallel edges) between the same pair of vertices are allowed.
- **(c) $K_n$:** Complete graph on $n$ vertices — every pair of distinct vertices is connected by exactly one edge. $|E(K_n)|=\binom{n}{2}=\frac{n(n-1)}{2}$.

---

**Q5.** How many edges does $K_5$ have?

**Solution:**
$$|E(K_5)| = \binom{5}{2} = \frac{5\times4}{2} = 10$$

---

**Q6.** Define a path and a cycle.

**Solution:**
- **Path:** A sequence of distinct vertices $v_0,v_1,\ldots,v_k$ where each consecutive pair $\{v_{i},v_{i+1}\}$ is an edge. Length = number of edges = $k$.
- **Cycle:** A closed path — a sequence $v_0,v_1,\ldots,v_k,v_0$ where $v_0=v_k$ and all other vertices are distinct.

---

**Q7.** Define a tree.

**Solution:**
A **tree** is a connected, acyclic (cycle-free) undirected graph. A tree with $n$ vertices has exactly $n-1$ edges.

---

**Q8.** What is a bipartite graph?

**Solution:**
A graph $G=(V,E)$ is **bipartite** if $V$ can be partitioned into two disjoint sets $V_1$ and $V_2$ such that every edge connects a vertex in $V_1$ to a vertex in $V_2$ (no edges within $V_1$ or within $V_2$).

---

**Q9.** State the condition for an Eulerian circuit to exist.

**Solution:**
An undirected connected graph has an **Eulerian circuit** (a closed walk traversing every edge exactly once) if and only if **every vertex has even degree**.

An **Eulerian path** (not necessarily closed) exists iff there are exactly **0 or 2** vertices of odd degree.

## 🟡 Medium (Q10–Q18)

**Q10.** Prove: In any graph, the number of odd-degree vertices is even.

**Solution:**
By the Handshaking Theorem: $\sum_{v}\deg(v) = 2|E|$ (even).

Let $S_{\text{odd}}$ = sum over odd-degree vertices, $S_{\text{even}}$ = sum over even-degree vertices.

$S_{\text{odd}} + S_{\text{even}} = 2|E|$ (even). Since $S_{\text{even}}$ is a sum of even numbers (hence even), $S_{\text{odd}}$ must also be even. The sum of an odd count of odd numbers would be odd, so the count of odd-degree vertices must be even. $\blacksquare$

---

**Q11.** State and explain Euler's Theorem for Eulerian paths and circuits.

**Solution:**
**Euler's Theorem:**
1. A connected graph has an **Eulerian circuit** ↔ every vertex has even degree.
2. A connected graph has an **Eulerian path** (not a circuit) ↔ exactly two vertices have odd degree (these are the start and end).

**Intuition:** Entering and leaving a vertex uses 2 edges; to have a circuit, no "leftover" edge can exist at any vertex.

**Q12.** How many edges does a spanning tree of $K_6$ have? How many spanning trees?

**Solution:**
$K_6$ has $n=6$ vertices. A spanning tree has $n-1 = \mathbf{5}$ edges.

Number of spanning trees by **Cayley's Formula**: $n^{n-2} = 6^4 = \mathbf{1296}$.

**Q13.** Apply Prim's algorithm to find the MST of: Vertices $\{A,B,C,D\}$, edges with weights: $AB=4$, $AC=2$, $BC=1$, $BD=5$, $CD=3$.

**Solution:**
Start at $A$. Visited: $\{A\}$.

1. Min edge from $\{A\}$: $AC=2$. Add. Visited: $\{A,C\}$.
2. Min edge from $\{A,C\}$: $BC=1$. Add. Visited: $\{A,B,C\}$.
3. Min edge from $\{A,B,C\}$: $CD=3$ or $BD=5$. Add $CD=3$. Visited: $\{A,B,C,D\}$.

**MST edges:** $\{AC, BC, CD\}$. **Total weight** $= 2+1+3 = \mathbf{6}$.

**Q14.** Apply Kruskal's algorithm to the graph in Q13.

**Solution:**
Sort edges: $BC=1$, $AC=2$, $CD=3$, $AB=4$, $BD=5$.

1. Add $BC=1$: $\{B,C\}$ — no cycle ✓.
2. Add $AC=2$: $\{A,B,C\}$ — no cycle ✓.
3. Add $CD=3$: $\{A,B,C,D\}$ — no cycle ✓.
4. ($AB=4$: $A,B$ already connected — skip.)

**MST:** $\{BC, AC, CD\}$, weight $= 1+2+3 = \mathbf{6}$. Same as Prim's. ✓

**Q15.** Define a Hamiltonian path and circuit. Does $K_3$ have one?

**Solution:**
- **Hamiltonian path:** A path that visits every vertex exactly once.
- **Hamiltonian circuit:** A closed path visiting every vertex exactly once.

**$K_3$ (triangle):** Vertices $\{1,2,3\}$, edges $\{12,23,13\}$.

Path: $1\to2\to3$ ✓ (Hamiltonian path). Circuit: $1\to2\to3\to1$ ✓ (Hamiltonian circuit). **Yes, $K_3$ has both.**

**Q16.** What is the chromatic number $\chi(G)$? Find $\chi(K_n)$ and $\chi(C_5)$.

**Solution:**
$\chi(G)$ = minimum number of colors needed to color vertices so no two adjacent vertices share a color.

- $\chi(K_n) = n$ (every vertex is adjacent to every other — needs $n$ colors).
- $\chi(C_5) = 3$ (odd cycle — needs 3 colors; 2 colors fail since any odd cycle is not 2-colorable).

---

**Q17.** Prove: A graph is bipartite iff it contains no odd-length cycles.

**Solution:**
**(⇒)** If $G$ is bipartite with parts $V_1,V_2$: any cycle alternates between $V_1$ and $V_2$. Each step alternates parts, so a cycle of length $k$ returns to start only if $k$ is even. No odd cycles. $\blacksquare$

**(⇐)** If $G$ has no odd cycles: BFS-color each component — color $v$ with color based on distance parity from source $s$ (0=even, 1=odd). Any edge $\{u,v\}$ connects vertices at different distances (by BFS). If BFS gives two odd-distance vertices connected → there'd be an odd cycle (contradiction). Hence the 2-coloring is valid and $G$ is bipartite. $\blacksquare$

**Q18.** Find all spanning trees of the graph $P_3$ (path on 3 vertices: $A-B-C$).

**Solution:**
$P_3$ has vertices $\{A,B,C\}$ and edges $\{AB, BC\}$.

A spanning tree needs 2 edges, connecting all 3 vertices without a cycle.

The only spanning tree is $P_3$ itself: $\{AB, BC\}$.

(There is no third edge to add anyway — the graph has only 2 edges.)

**1 spanning tree.**

## 🔴 Hard (Q19–Q25)

**Q19.** Prove a tree with $n$ vertices has exactly $n-1$ edges (by induction on $n$).

**Solution:**
**Base case:** $n=1$: one vertex, no edges, $1-1=0$ ✓.

**Inductive step:** Assume every tree with $k<n$ vertices has $k-1$ edges. Consider a tree $T$ with $n$ vertices. Since $T$ is finite and acyclic, it has at least one **leaf** (degree-1 vertex) $v$ (provable: otherwise, by pigeonhole, there's a cycle).

Remove $v$ and its incident edge. The resulting graph $T'$ is still connected (removing a leaf doesn't disconnect a tree) and acyclic. $T'$ has $n-1$ vertices, so by induction has $(n-1)-1=n-2$ edges.

$T$ has one more edge than $T'$: $(n-2)+1=n-1$ edges. $\blacksquare$

---

**Q20.** Prove: A connected graph $G$ with $n$ vertices is a tree ↔ it has $n-1$ edges.

**Solution:**
**(⇒ Tree $\implies$ $n-1$ edges):** By Q19. $\blacksquare$

**(⇐ Connected + $n-1$ edges $\implies$ Tree):** Suppose $G$ is connected with $n-1$ edges but has a cycle. Remove one edge from the cycle — the graph remains connected (since the cycle provided an alternate path). The resulting graph $G'$ is connected with $n-2$ edges and $n$ vertices.

But a connected graph needs at least $n-1$ edges (spanning tree bound). Contradiction. Hence $G$ is acyclic → $G$ is a tree. $\blacksquare$

**Q21.** State and prove the Four-Color Theorem (statement only) and prove $\chi(G)\leq\Delta(G)+1$ (Brooks' bound).

**Solution:**
**Four-Color Theorem:** Every planar graph has chromatic number $\leq4$. (Proof is computer-assisted, beyond scope.)

**Greedy bound $\chi(G)\leq\Delta(G)+1$:**
Order vertices $v_1,v_2,\ldots,v_n$ arbitrarily. Color greedily: assign $v_i$ the smallest color not used by its already-colored neighbors.

At step $i$, vertex $v_i$ has at most $\Delta(G)$ neighbors, of which at most $\Delta(G)$ are already colored → at most $\Delta(G)$ forbidden colors. With $\Delta(G)+1$ colors available, there is always at least 1 available color.

Therefore, greedy coloring uses at most $\Delta(G)+1$ colors. $\blacksquare$

**Q22.** A complete bipartite graph $K_{m,n}$: find $|E|$, $\chi$, and whether it has an Eulerian circuit.

**Solution:**
- **$|E(K_{m,n})|$** $= mn$ (each of $m$ vertices in $V_1$ connects to all $n$ in $V_2$).
- **$\chi(K_{m,n})$** $= 2$ (bipartite → 2-colorable, and there's at least one edge → needs at least 2).
- **Eulerian circuit:** Every vertex in $V_1$ has degree $n$; every vertex in $V_2$ has degree $m$. All degrees even ↔ both $m$ and $n$ are even. So Eulerian circuit exists iff **$m$ and $n$ are both even**.

**Q23.** Prove Ore's Theorem: If $\deg(u)+\deg(v)\geq n$ for every pair of non-adjacent $u,v$, then $G$ has a Hamiltonian circuit.

**Solution:**
**Sketch:** By contradiction. Suppose $G$ satisfies the condition but has no Hamiltonian circuit. Add edges until $G'$ is maximal (adding any edge creates a Hamiltonian circuit). In $G'$, there's a Hamiltonian **path** $v_1,v_2,\ldots,v_n$.

Since $v_1v_n$ is not an edge (else we'd have circuit), $\deg(v_1)+\deg(v_n)\geq n$.

Let $A=\{i:v_1v_{i+1}\in E(G')\}$ and $B=\{i:v_nv_i\in E(G')\}$. $|A|=\deg(v_1)\geq n-\deg(v_n)$. Both subsets of $\{1,\ldots,n-1\}$, so $|A|+|B|\geq n>n-1$. By pigeonhole, $\exists j\in A\cap B$: $v_1v_{j+1}$ and $v_nv_j$ are edges. Then $v_1\to v_{j+1}\to\cdots\to v_n\to v_j\to v_{j-1}\to\cdots\to v_1$ is a Hamiltonian circuit. Contradiction. $\blacksquare$

**Q24.** Apply Dijkstra's algorithm: $s=A$, vertices $\{A,B,C,D,E\}$, edges: $AB=4$, $AC=2$, $BC=1$, $BD=5$, $CD=8$, $DE=2$, $BE=3$.

**Solution:**
Initialize: $d(A)=0$, all others $=\infty$.

**Step 1:** Visit $A$ (min dist=0). Update: $d(B)=4$, $d(C)=2$.

**Step 2:** Visit $C$ (dist=2). Update: $d(B)=\min(4,2+1)=3$, $d(D)=\min(\infty,2+8)=10$.

**Step 3:** Visit $B$ (dist=3). Update: $d(D)=\min(10,3+5)=8$, $d(E)=\min(\infty,3+3)=6$.

**Step 4:** Visit $E$ (dist=6). Update: $d(D)=\min(8,6+2)=8$.

**Step 5:** Visit $D$ (dist=8).

**Shortest paths from $A$:**
- $A\to B$: 3 (via $C$), $A\to C$: 2, $A\to D$: 8 (via $C,B$), $A\to E$: 6 (via $C,B$).

**Q25.** Prove: Every connected graph has a spanning tree.

**Solution:**
**Proof by edge removal:**

Let $G$ be a connected graph. If $G$ is already a tree, done.

Otherwise, $G$ contains a cycle $C$. Remove any edge $e\in C$. The resulting graph $G'=G-e$ is still connected (since the removed edge was part of a cycle, the other path around the cycle still connects the endpoints of $e$). $G'$ has one fewer edge.

Repeat: if $G'$ contains a cycle, remove an edge. Since $|E|$ is finite and decreases each step, this terminates.

When no cycle remains, we have a connected acyclic graph — a **spanning tree**. $\blacksquare$
