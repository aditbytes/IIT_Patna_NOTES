## 🟢 Easy (Q1–Q9)

**Q1.** Define a Finite State Machine (FSM). What are its components?

**Solution:**
A **Finite State Machine** (FSM) or **Finite Automaton** is a 5-tuple $M=(Q,\Sigma,\delta,q_0,F)$ where:
- $Q$ = finite set of **states**
- $\Sigma$ = **input alphabet**
- $\delta: Q\times\Sigma\to Q$ = **transition function**
- $q_0\in Q$ = **start state**
- $F\subseteq Q$ = set of **accepting states** (final states)

---

**Q2.** What is the difference between a DFA and an NFA?

**Solution:**
| Feature | DFA | NFA |
|---------|-----|-----|
| Transitions | Exactly one per $(q,a)$ | Zero, one, or more per $(q,a)$ |
| $\varepsilon$-transitions | Not allowed | Allowed (in NFA-$\varepsilon$) |
| Computation | Deterministic | Non-deterministic |
| Equivalence | Equal expressive power | Same as DFA |

Every NFA can be converted to an equivalent DFA (subset construction).

---

**Q3.** Draw a DFA that accepts strings over $\{0,1\}$ ending in 0.

**Solution:**
States: $q_0$ (last bit was 1 or start), $q_1$ (last bit was 0, accepting).

Transitions:
- $\delta(q_0,0)=q_1$, $\delta(q_0,1)=q_0$
- $\delta(q_1,0)=q_1$, $\delta(q_1,1)=q_0$

Start: $q_0$. Accept: $\{q_1\}$.

**State diagram:** $q_0\xrightarrow{0}q_1\xrightarrow{1}q_0$, $q_0\xrightarrow{1}q_0$, $q_1\xrightarrow{0}q_1$.

---

**Q4.** What is a transition table? Write one for the DFA in Q3.

**Solution:**
| State | Input 0 | Input 1 |
|-------|---------|---------|
| $\to q_0$ | $q_1$ | $q_0$ |
| $*q_1$ | $q_1$ | $q_0$ |

$\to$ = start state, $*$ = accepting state.

**Q5.** What language does the following DFA accept?
States: $\{A,B\}$; $A$ is start and accepting; $\delta(A,a)=B$, $\delta(A,b)=A$, $\delta(B,a)=A$, $\delta(B,b)=B$.

**Solution:**
Trace: $A$ accepts iff we return to $A$.

$A\xrightarrow{a}B\xrightarrow{a}A$: "aa" accepted ✓.
$A\xrightarrow{b}A$: "b" accepted ✓.
$A\xrightarrow{a}B$: "a" rejected (not in $F$).

This DFA accepts strings with an **even number of $a$'s** (regardless of $b$'s). Language: $L=\{w\in\{a,b\}^* : |w|_a\text{ is even}\}$.

---

**Q6.** Define an accepting computation and a rejecting computation.

**Solution:**
Given DFA $M$ and input string $w=a_1a_2\ldots a_n$:
- Start: state $q_0$.
- Transition: $q_0\xrightarrow{a_1}q_1\xrightarrow{a_2}\cdots\xrightarrow{a_n}q_n$.
- **Accepting:** If $q_n\in F$ (final state is accepting).
- **Rejecting:** If $q_n\notin F$ (final state is not accepting).

---

**Q7.** What is $\varepsilon$ (epsilon) in automata theory? What is $\varepsilon^*$?

**Solution:**
- $\varepsilon$ = the **empty string** (string of length 0).
- $\varepsilon^*$ = Kleene star of $\varepsilon$ = $\{\varepsilon\}$ (only the empty string).
- In NFAs, an $\varepsilon$-**transition** allows the automaton to change state without consuming input.

---

**Q8.** What is the language of an FSM?

**Solution:**
The **language** $L(M)$ of FSM $M$ is the set of all strings that $M$ accepts:
$$L(M) = \{w\in\Sigma^* : \hat{\delta}(q_0,w)\in F\}$$
where $\hat{\delta}$ is the extended transition function (applied to strings).

---

**Q9.** Define Mealy machine and Moore machine.

**Solution:**
- **Moore machine:** Output depends only on the **current state**. Output function $\lambda:Q\to\Gamma$.
- **Mealy machine:** Output depends on both the **current state and input**. Output function $\lambda:Q\times\Sigma\to\Gamma$.

Both are transducers (produce output for each input). They are equivalent in expressive power.

## 🟡 Medium (Q10–Q18)

**Q10.** Trace the DFA from Q3 on input "1001".

**Solution:**
Start: $q_0$.
- Read 1: $\delta(q_0,1)=q_0$. State: $q_0$.
- Read 0: $\delta(q_0,0)=q_1$. State: $q_1$.
- Read 0: $\delta(q_1,0)=q_1$. State: $q_1$.
- Read 1: $\delta(q_1,1)=q_0$. State: $q_0$.

Final state $q_0\notin F$. **"1001" is rejected.** (Ends in 1, not 0.)

**Q11.** Design a DFA over $\{0,1\}$ that accepts strings divisible by 3 (binary).

**Solution:**
States represent remainder mod 3: $\{q_0,q_1,q_2\}$ where $q_i$ = remainder is $i$.

Transitions (if current number is $n$ and we read bit $b$, new value is $2n+b$):
- $\delta(q_0,0)=q_0$ ($2\times0+0=0$), $\delta(q_0,1)=q_1$ ($0+1=1$)
- $\delta(q_1,0)=q_2$ ($2\times1=2$), $\delta(q_1,1)=q_0$ ($2+1=3\equiv0$)
- $\delta(q_2,0)=q_1$ ($2\times2=4\equiv1$), $\delta(q_2,1)=q_2$ ($4+1=5\equiv2$)

Start: $q_0$. Accept: $\{q_0\}$ (remainder 0 = divisible by 3). Also accept empty string $\varepsilon$ (representing 0).

**Verify:** "110" = 6. $q_0\xrightarrow{1}q_1\xrightarrow{1}q_0\xrightarrow{0}q_0$. Accepted ✓.

**Q12.** Convert the NFA to a DFA using subset construction:
States $\{p,q,r\}$; start $p$; accept $\{r\}$; $\delta(p,a)=\{p,q\}$, $\delta(p,b)=\{q\}$, $\delta(q,a)=\{r\}$, $\delta(q,b)=\emptyset$, $\delta(r,a)=\emptyset$, $\delta(r,b)=\emptyset$.

**Solution:**
DFA states are subsets of $\{p,q,r\}$.

Start: $\{p\}$.

| DFA State | On $a$ | On $b$ |
|-----------|--------|--------|
| $\{p\}$ | $\{p,q\}$ | $\{q\}$ |
| $\{p,q\}$ | $\{p,q,r\}$ | $\{q\}$ |
| $\{q\}$ | $\{r\}$ | $\emptyset$ |
| $\{p,q,r\}$ | $\{p,q,r\}$ | $\{q\}$ |
| $\{r\}$ | $\emptyset$ | $\emptyset$ |
| $\emptyset$ | $\emptyset$ | $\emptyset$ |

Accept states (containing $r$): $\{p,q,r\}$, $\{r\}$. Language: strings with a $q$ after first $p$ followed by $a$ — i.e., strings containing "aa" as substring.

**Q13.** What is the extended transition function $\hat{\delta}$?

**Solution:**
$\hat{\delta}: Q\times\Sigma^*\to Q$ processes an entire string:
$$\hat{\delta}(q,\varepsilon) = q$$
$$\hat{\delta}(q,wa) = \delta(\hat{\delta}(q,w),a) \quad\forall w\in\Sigma^*, a\in\Sigma$$

It extends $\delta$ from single characters to strings by recursive application.

**Q14.** Design a Moore machine to compute 1's complement of a binary string.

**Solution:**
Output each bit flipped: 0→1, 1→0.

**States:** Single state $q_0$ (no memory needed).

**Outputs:** $\lambda(q_0)$ with input 0 → output 1; with input 1 → output 0.

(For Moore machine, output is per-state, so we need two states.)

States: $q_0$ (initial, output arbitrary), $q_1$ (output 0), $q_0'$ (output 1)?

Actually, simplest Moore machine: One state $q_0$ for each output. Since output depends on next state after reading:

States: $\{q_\text{start},q_0,q_1\}$ where entering $q_0$ means last input was 0 (output 1), entering $q_1$ means last input was 1 (output 0).

$\lambda(q_0)=1$, $\lambda(q_1)=0$. Transitions from any state: on 0 → $q_0$, on 1 → $q_1$.

**Q15.** State the pumping lemma for regular languages (statement only) and use it to show $\{0^n1^n : n\geq0\}$ is not regular.

**Solution:**
**Pumping Lemma:** If $L$ is regular, $\exists$ pumping length $p$ such that every $w\in L$ with $|w|\geq p$ can be split as $w=xyz$ where: (1) $|xy|\leq p$, (2) $|y|\geq1$, (3) $xy^iz\in L$ for all $i\geq0$.

**Proof $L=\{0^n1^n\}$ is not regular:**
Suppose $L$ is regular with pumping length $p$. Take $w=0^p1^p\in L$.

By pumping lemma: $w=xyz$ with $|xy|\leq p$ → $y=0^k$ for some $k\geq1$ (all 0's).

$xy^2z = 0^{p+k}1^p$ — has more 0's than 1's → $\notin L$. Contradiction. $\therefore L$ is **not regular**. $\blacksquare$

**Q16.** Minimize the DFA: States $\{A,B,C,D,E\}$, start $A$, accept $\{C,E\}$:
$\delta(A,0)=B$,$\delta(A,1)=C$,$\delta(B,0)=B$,$\delta(B,1)=D$,$\delta(C,0)=B$,$\delta(C,1)=C$,$\delta(D,0)=B$,$\delta(D,1)=E$,$\delta(E,0)=B$,$\delta(E,1)=C$.

**Solution:**
**Step 1:** Partition: $\Pi_0 = \{\{C,E\},\{A,B,D\}\}$ (accepting vs. non-accepting).

**Step 2:** Refine using transitions.

Check $\{A,B,D\}$ with input 1: $\delta(A,1)=C\in\{C,E\}$; $\delta(B,1)=D\in\{A,B,D\}$; $\delta(D,1)=E\in\{C,E\}$.

$A$ and $D$ go to accepting, $B$ goes to non-accepting. Split: $\{A,D\}$ and $\{B\}$.

Check $\{A,D\}$ with input 0: $\delta(A,0)=B$, $\delta(D,0)=B$. Same. With 1: $\delta(A,1)=C$, $\delta(D,1)=E$. Both in $\{C,E\}$. $A$ and $D$ behave identically.

Check $\{C,E\}$: $\delta(C,0)=B$, $\delta(E,0)=B$. $\delta(C,1)=C$, $\delta(E,1)=C$. Identical!

**Minimized DFA:** 3 states: $\{A,D\}$, $\{B\}$, $\{C,E\}$.

**Q17.** What is the regular expression for the language of DFA in Q3 (strings ending in 0)?

**Solution:**
Strings ending in 0 over $\{0,1\}$: any string of 0's and 1's, ending in 0.

Regular expression: $(0+1)^*0$ or equivalently $\Sigma^*0$ where $\Sigma=\{0,1\}$.

Expanded: any prefix (including empty) followed by 0: $\varepsilon0 + (0+1)(0+1)^*0$. Simplest form: **$(0+1)^*0$**.

**Q18.** Find the number of strings of length $n$ accepted by the DFA from Q5 (even number of $a$'s).

**Solution:**
Alphabet $\{a,b\}$. A string of length $n$ is accepted iff it has an **even number** of $a$'s.

Number of $a$'s can be $0,2,4,\ldots$. Number of strings with exactly $k$ $a$'s (rest $b$'s): $\binom{n}{k}$.

Count $=\sum_{k\text{ even}}\binom{n}{k} = 2^{n-1}$ (by the identity $\sum_{k\text{ even}}\binom{n}{k}=2^{n-1}$).

**Number of accepted strings of length $n$ = $2^{n-1}$.**

**Q19.** Construct an NFA (with $\varepsilon$-transitions) for the regular expression $(ab)^*+a^*$.

**Solution:**
Use Thompson's construction:

For $(ab)^*$: $q_0\xrightarrow{\varepsilon}q_1\xrightarrow{a}q_2\xrightarrow{b}q_3\xrightarrow{\varepsilon}q_1$ (loop), $q_0\xrightarrow{\varepsilon}q_4$ (skip for zero repetitions), $q_3\xrightarrow{\varepsilon}q_4$.

For $a^*$: $q_0'\xrightarrow{\varepsilon}q_5\xrightarrow{a}q_5$ (loop), $q_0'\xrightarrow{\varepsilon}q_6$.

Combine with union: New start $s_0\xrightarrow{\varepsilon}q_0$ and $s_0\xrightarrow{\varepsilon}q_0'$. Both thread to new accept state $f$.

(Full diagram is complex; the key is applying Thompson's NFA construction rules recursively.)

**Q20.** Prove that DFA and NFA recognize the same class of languages (regular languages).

**Solution:**
**DFA → NFA:** Every DFA is trivially an NFA (single transition per state/input).

**NFA → DFA (Subset Construction):** For NFA $N=(Q,\Sigma,\delta,q_0,F)$, construct DFA $D=(2^Q,\Sigma,\delta',\{q_0\},F')$ where:
- $\delta'(S,a) = \bigcup_{q\in S}\delta(q,a)$ for $S\subseteq Q$.
- $F' = \{S\subseteq Q : S\cap F\neq\emptyset\}$.

**Claim:** $D$ accepts $w$ iff $N$ accepts $w$.

**Proof by induction on $|w|$:** Base: $\hat{\delta}'(\{q_0\},\varepsilon)=\{q_0\}=\hat{\delta}(q_0,\varepsilon)$. Step: $\hat{\delta}'(S,wa)=\delta'(\hat{\delta}'(S,w),a)=\bigcup_{q\in\hat{\delta}(q_0,w)}\delta(q,a)=\hat{\delta}(q_0,wa)$.

So $D$ accepts $w$ iff final state $\in F'$ iff $\hat{\delta}(q_0,w)\cap F\neq\emptyset$ iff $N$ accepts $w$. $\blacksquare$

**Q21.** Show that the class of regular languages is closed under (i) union, (ii) concatenation, (iii) Kleene star.

**Solution:**
Let $L_1,L_2$ be regular, accepted by DFAs/NFAs $M_1,M_2$.

**(i) Union:** Construct NFA: new start $s_0\xrightarrow{\varepsilon}q_0^{(1)}$ and $s_0\xrightarrow{\varepsilon}q_0^{(2)}$. Accept states of $M_1\cup F_2$. NFA accepts $w\in L_1\cup L_2$. Convert to DFA. $\checkmark$

**(ii) Concatenation:** New start $= q_0^{(1)}$. Add $\varepsilon$-transitions from each $f\in F_1$ to $q_0^{(2)}$. Accept states $= F_2$. $\checkmark$

**(iii) Kleene Star:** New start $s_0$ (also accepting). $\varepsilon$-transition $s_0\to q_0^{(1)}$. Each $f\in F_1$ has $\varepsilon$-transition back to $q_0^{(1)}$. $\checkmark$ $\blacksquare$

**Q22.** Myhill-Nerode Theorem: State and use it to show $L=\{a^nb^n : n\geq0\}$ is not regular.

**Solution:**
**Myhill-Nerode:** $L$ is regular iff the number of equivalence classes of $\sim_L$ (Nerode relation: $x\sim_L y$ if $\forall z: xz\in L\Leftrightarrow yz\in L$) is finite.

**For $L=\{a^nb^n\}$:** Consider $x_i=a^i$ for $i=0,1,2,\ldots$. Claim all $x_i$ are in different equivalence classes.

For $i\neq j$ (say $i<j$): take $z=b^i$. Then $x_iz=a^ib^i\in L$ but $x_jz=a^jb^i\notin L$ (since $j\neq i$).

So $x_i\not\sim_L x_j$ for $i\neq j$ → infinitely many classes → $L$ is **not regular**. $\blacksquare$

**Q23.** Design a 2-state Moore machine that outputs 1 when it has seen an odd number of 1's (input alphabet $\{0,1\}$).

**Solution:**
States: $q_0$ (even count of 1's, output 0), $q_1$ (odd count, output 1).

**Output function:** $\lambda(q_0)=0$, $\lambda(q_1)=1$.

**Transition function:**
| State | On 0 | On 1 |
|-------|------|------|
| $q_0$ | $q_0$ | $q_1$ |
| $q_1$ | $q_1$ | $q_0$ |

Start: $q_0$. Output is $\lambda$ of current state (output produced on entering state, or on each step depending on convention).

**Q24.** Convert NFA-$\varepsilon$ to NFA: Use $\varepsilon$-closure.

NFA-$\varepsilon$: $Q=\{0,1,2\}$, $\Sigma=\{a,b\}$, start=0, accept$=\{2\}$.
$\delta(0,a)=\{1\}$, $\delta(1,b)=\{2\}$, $\delta(0,\varepsilon)=\{1\}$.

**Solution:**
**$\varepsilon$-closures:**
$\varepsilon\text{-cl}(0)=\{0,1\}$ (0 can reach 1 via $\varepsilon$).
$\varepsilon\text{-cl}(1)=\{1\}$. $\varepsilon\text{-cl}(2)=\{2\}$.

**NFA without $\varepsilon$ transitions** (states are $\varepsilon$-closures):
Start state: $\varepsilon\text{-cl}(0)=\{0,1\}$.

$\delta'(\{0,1\},a)=\varepsilon\text{-cl}(\delta(0,a)\cup\delta(1,a))=\varepsilon\text{-cl}(\{1\}\cup\emptyset)=\{1\}$.
$\delta'(\{0,1\},b)=\varepsilon\text{-cl}(\delta(0,b)\cup\delta(1,b))=\varepsilon\text{-cl}(\emptyset\cup\{2\})=\{2\}$.
$\delta'(\{1\},a)=\varepsilon\text{-cl}(\emptyset)=\emptyset$.
$\delta'(\{1\},b)=\varepsilon\text{-cl}(\{2\})=\{2\}$.
$\delta'(\{2\},a)=\delta'(\{2\},b)=\emptyset$.

Language accepted: $\{b, ab\}$ (any string leading to $\{2\}$).

**Q25.** Prove the Pumping Lemma for regular languages.

**Solution:**
Let $L$ be regular, accepted by DFA $M$ with $p=|Q|$ states. Let $w\in L$ with $|w|\geq p$.

When $M$ processes $w=a_1a_2\ldots a_n$ ($n\geq p$), it visits states $q_0,q_1,\ldots,q_n$ (n+1 states in sequence). Since $n\geq p=|Q|$, by the **Pigeonhole Principle**, among $q_0,q_1,\ldots,q_p$ ($p+1$ states), some state repeats: $q_i=q_j$ for $0\leq i<j\leq p$.

Define $x=a_1\ldots a_i$, $y=a_{i+1}\ldots a_j$, $z=a_{j+1}\ldots a_n$.

- $|xy|=j\leq p$ ✓ (since $j\leq p$).
- $|y|=j-i\geq1$ ✓ (since $i<j$).
- $xy^kz$: from $q_0$, process $x$ reaches $q_i=q_j$. Looping $y$ any number of times stays at $q_i=q_j$ (it's a cycle). Then $z$ takes us from $q_j$ to the same final state (still accepting). So $xy^kz\in L$ for all $k\geq0$. $\blacksquare$
