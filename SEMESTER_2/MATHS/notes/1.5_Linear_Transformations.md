## What is a Linear Transformation? (Layman's Explanation)

Imagine you have a **machine** that takes a vector as input and gives another vector as output. If this machine follows two simple rules ÔÇö (1) processing two vectors separately and adding the results is the same as adding the vectors first and then processing, and (2) stretching a vector first then processing is the same as processing then stretching ÔÇö then this machine is a **linear transformation**.

---

## Formal Definition

A function $T: V \to W$ (from vector space $V$ to vector space $W$) is a **linear transformation** if for all $u, v \in V$ and all scalars $\alpha, \beta \in F$:

$$
T(\alpha u + \beta v) = \alpha T(u) + \beta T(v)
$$

This is equivalent to satisfying BOTH:
1. **Additivity:** $T(u + v) = T(u) + T(v)$
2. **Homogeneity:** $T(\alpha v) = \alpha T(v)$

### Important Consequences
- $T(\mathbf{0}) = \mathbf{0}$ (zero goes to zero ÔÇö always!)
- $T(-v) = -T(v)$
- $T(\alpha_1 v_1 + \alpha_2 v_2 + \cdots + \alpha_n v_n) = \alpha_1 T(v_1) + \alpha_2 T(v_2) + \cdots + \alpha_n T(v_n)$

## Examples of Linear Transformations

### Example 1: $T : \mathbb{R}^2 \to \mathbb{R}^2$ defined by $T(x, y) = (2x + y, x - y)$

**Check linearity:**

Let $u = (x_1, y_1)$, $v = (x_2, y_2)$.

$T(u + v) = T(x_1+x_2, y_1+y_2) = (2(x_1+x_2)+(y_1+y_2), (x_1+x_2)-(y_1+y_2))$

$= (2x_1+y_1, x_1-y_1) + (2x_2+y_2, x_2-y_2) = T(u) + T(v)$ Ôťô

$T(\alpha u) = T(\alpha x_1, \alpha y_1) = (2\alpha x_1 + \alpha y_1, \alpha x_1 - \alpha y_1) = \alpha(2x_1+y_1, x_1-y_1) = \alpha T(u)$ Ôťô

**Linear transformation** Ôťô

### Example 2: $T : \mathbb{R}^3 \to \mathbb{R}^2$ defined by $T(x, y, z) = (x + y, y + z)$

$T(\alpha(x_1,y_1,z_1) + \beta(x_2,y_2,z_2))$
$= T(\alpha x_1 + \beta x_2, \alpha y_1 + \beta y_2, \alpha z_1 + \beta z_2)$
$= (\alpha x_1 + \beta x_2 + \alpha y_1 + \beta y_2, \alpha y_1 + \beta y_2 + \alpha z_1 + \beta z_2)$
$= \alpha(x_1+y_1, y_1+z_1) + \beta(x_2+y_2, y_2+z_2)$
$= \alpha T(x_1,y_1,z_1) + \beta T(x_2,y_2,z_2)$ Ôťô

### Example 3: NOT a linear transformation

$T(x, y) = (x + 1, y)$

$T(0, 0) = (1, 0) \neq (0, 0)$

Since $T(\mathbf{0}) \neq \mathbf{0}$, this is **NOT** a linear transformation.

### Example 4: NOT a linear transformation

$T(x, y) = (x^2, y)$

$T(2 \cdot (1, 0)) = T(2, 0) = (4, 0)$

$2 \cdot T(1, 0) = 2(1, 0) = (2, 0)$

$(4, 0) \neq (2, 0)$ Ôćĺ **NOT** a linear transformation.

### Common Linear Transformations

| Transformation | Formula | From Ôćĺ To |
|---------------|---------|-----------|
| **Zero transformation** | $T(v) = \mathbf{0}$ | $V \to W$ |
| **Identity transformation** | $T(v) = v$ | $V \to V$ |
| **Scaling** | $T(v) = cv$ | $V \to V$ |
| **Rotation** by $\theta$ in $\mathbb{R}^2$ | $T(x,y) = (x\cos\theta - y\sin\theta, x\sin\theta + y\cos\theta)$ | $\mathbb{R}^2 \to \mathbb{R}^2$ |
| **Projection** onto $x$-axis | $T(x, y) = (x, 0)$ | $\mathbb{R}^2 \to \mathbb{R}^2$ |
| **Reflection** about $x$-axis | $T(x, y) = (x, -y)$ | $\mathbb{R}^2 \to \mathbb{R}^2$ |
| **Differentiation** | $T(p(x)) = p'(x)$ | $P_n \to P_{n-1}$ |

## Kernel (Null Space) of a Linear Transformation

### Definition
The **kernel** (or **null space**) of $T: V \to W$ is the set of all vectors in $V$ that map to the **zero vector** in $W$:

$$
\ker(T) = \{v \in V : T(v) = \mathbf{0}\}
$$

**In simple words:** Kernel = everything that $T$ "kills" (sends to zero).

### Key Fact
$\ker(T)$ is always a **subspace** of $V$.

### Example 1: Find $\ker(T)$

$T: \mathbb{R}^3 \to \mathbb{R}^2$, $T(x, y, z) = (x + y, y + z)$

Set $T(x, y, z) = (0, 0)$:
- $x + y = 0 \Rightarrow x = -y$
- $y + z = 0 \Rightarrow z = -y$

Let $y = t$: $(x, y, z) = (-t, t, -t) = t(-1, 1, -1)$

$\ker(T) = \text{span}\{(-1, 1, -1)\}$, dimension = **1**.

### Example 2: Find $\ker(T)$

$T: \mathbb{R}^2 \to \mathbb{R}^2$, $T(x, y) = (2x + y, x - y)$

Set $T(x, y) = (0, 0)$:
- $2x + y = 0$
- $x - y = 0 \Rightarrow x = y$

Substituting: $2y + y = 0 \Rightarrow 3y = 0 \Rightarrow y = 0, x = 0$.

$\ker(T) = \{\mathbf{0}\}$, dimension = **0**.

(This means $T$ is **one-to-one / injective**.)

## Range (Image) of a Linear Transformation

### Definition
The **range** (or **image**) of $T: V \to W$ is the set of all vectors in $W$ that are "hit" by $T$:

$$
\text{Range}(T) = \{T(v) : v \in V\} = \{w \in W : w = T(v) \text{ for some } v \in V\}
$$

**In simple words:** Range = everything that $T$ can produce as output.

### Key Fact
$\text{Range}(T)$ is always a **subspace** of $W$.

### Example: Find Range

$T: \mathbb{R}^3 \to \mathbb{R}^2$, $T(x, y, z) = (x + y, y + z)$

$T(x, y, z) = (x + y, y + z) = x(1, 0) + y(1, 1) + z(0, 1)$

Range$(T) = \text{span}\{(1, 0), (1, 1), (0, 1)\}$

Since $(1, 0)$ and $(0, 1)$ are linearly independent and span $\mathbb{R}^2$:

Range$(T) = \mathbb{R}^2$, dimension = **2**.

(Since Range = entire codomain, $T$ is **onto / surjective**.)

## Rank and Nullity of a Linear Transformation

### Definitions
- **Nullity of $T$** = $\dim(\ker(T))$ = dimension of the kernel
- **Rank of $T$** = $\dim(\text{Range}(T))$ = dimension of the range

### Rank-Nullity Theorem (MOST IMPORTANT THEOREM)

$$
\boxed{\text{Rank}(T) + \text{Nullity}(T) = \dim(V)}
$$

Or equivalently:
$$
\dim(\text{Range}(T)) + \dim(\ker(T)) = \dim(V)
$$

Where $V$ is the **domain** of $T$.

### Example: Verify Rank-Nullity Theorem

$T: \mathbb{R}^3 \to \mathbb{R}^2$, $T(x, y, z) = (x + y, y + z)$

From earlier:
- Nullity = $\dim(\ker(T)) = 1$
- Rank = $\dim(\text{Range}(T)) = 2$

Check: $1 + 2 = 3 = \dim(\mathbb{R}^3)$ Ôťô

### Consequences of Rank-Nullity Theorem

For $T: V \to W$ with $\dim(V) = n$:

| Condition | Meaning |
|-----------|---------|
| Nullity = 0 | $T$ is **one-to-one** (injective) |
| Rank = $\dim(W)$ | $T$ is **onto** (surjective) |
| $n = \dim(W)$ and Nullity = 0 | $T$ is **bijective** (isomorphism) |

## Matrix Representation of Linear Transformation

Every linear transformation $T: \mathbb{R}^n \to \mathbb{R}^m$ can be represented by an $m \times n$ matrix $A$ such that:

$$
T(X) = AX
$$

### How to Find the Matrix

If $\{e_1, e_2, \ldots, e_n\}$ is the standard basis of $\mathbb{R}^n$, then:

$$
A = [T(e_1) \mid T(e_2) \mid \cdots \mid T(e_n)]
$$

The columns of $A$ are the images of the basis vectors.

### Example: Find the matrix of $T(x, y) = (2x + y, x - y, 3x)$

$T(1, 0) = (2, 1, 3)$ and $T(0, 1) = (1, -1, 0)$

$$
A = \begin{bmatrix} 2 & 1 \\ 1 & -1 \\ 3 & 0 \end{bmatrix}
$$

Verify: $A\begin{bmatrix} x \\ y \end{bmatrix} = \begin{bmatrix} 2x+y \\ x-y \\ 3x \end{bmatrix} = T(x, y)$ Ôťô

### Example: Matrix of rotation by angle $\theta$ in $\mathbb{R}^2$

$T(1, 0) = (\cos\theta, \sin\theta)$ and $T(0, 1) = (-\sin\theta, \cos\theta)$

$$
A = \begin{bmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{bmatrix}
$$

## One-to-One and Onto

### One-to-One (Injective)
$T$ is one-to-one if $T(u) = T(v) \Rightarrow u = v$.

**Equivalent condition:** $\ker(T) = \{\mathbf{0}\}$ (nullity = 0).

### Onto (Surjective)
$T$ is onto if for every $w \in W$, there exists $v \in V$ such that $T(v) = w$.

**Equivalent condition:** Range$(T) = W$ (rank = $\dim(W)$).

### Isomorphism
$T$ is an **isomorphism** if it is both one-to-one and onto.

Two vector spaces are **isomorphic** ($V \cong W$) if there exists an isomorphism between them.

**Key fact:** Two finite-dimensional vector spaces are isomorphic **if and only if they have the same dimension**.

---

## Composition of Linear Transformations

If $T_1: V \to W$ and $T_2: W \to U$ are linear transformations, then:

$$
(T_2 \circ T_1)(v) = T_2(T_1(v))
$$

is also a linear transformation from $V$ to $U$.

**Matrix representation:** If $T_1$ has matrix $A$ and $T_2$ has matrix $B$, then $T_2 \circ T_1$ has matrix $BA$.

## Practice Questions

**Q1.** Is $T: \mathbb{R}^2 \to \mathbb{R}^3$ defined by $T(x, y) = (x + y, x, 2y)$ a linear transformation? Find its kernel and range.

**Solution:**
$T(\alpha(x_1,y_1) + \beta(x_2,y_2)) = T(\alpha x_1+\beta x_2, \alpha y_1+\beta y_2)$
$= (\alpha x_1+\beta x_2+\alpha y_1+\beta y_2, \alpha x_1+\beta x_2, 2\alpha y_1+2\beta y_2)$
$= \alpha(x_1+y_1, x_1, 2y_1) + \beta(x_2+y_2, x_2, 2y_2) = \alpha T(x_1,y_1) + \beta T(x_2,y_2)$ Ôťô

**Kernel:** Set $T(x,y) = (0,0,0)$: $x+y=0$, $x=0$, $2y=0$ Ôćĺ $x=0, y=0$.
$\ker(T) = \{(0,0)\}$, nullity = 0. (T is one-to-one.)

**Range:** $T(x,y) = x(1,1,0) + y(1,0,2)$
Range = $\text{span}\{(1,1,0), (1,0,2)\}$, rank = 2.

Check: $0 + 2 = 2 = \dim(\mathbb{R}^2)$ Ôťô

**Q2.** Let $T: \mathbb{R}^3 \to \mathbb{R}^3$ be defined by $T(x, y, z) = (x + y + z, 2x + y, x - z)$. Find the matrix of $T$ and determine if $T$ is invertible.

**Solution:**
$T(1,0,0) = (1,2,1)$, $T(0,1,0) = (1,1,0)$, $T(0,0,1) = (1,0,-1)$

$$
A = \begin{bmatrix} 1 & 1 & 1 \\ 2 & 1 & 0 \\ 1 & 0 & -1 \end{bmatrix}
$$

$|A| = 1(-1-0) - 1(-2-0) + 1(0-1) = -1 + 2 - 1 = 0$

Since $|A| = 0$, $T$ is **NOT invertible** (not one-to-one, not onto).

**Q3.** Find the rank and nullity of $T: P_2 \to P_1$ defined by $T(p(x)) = p'(x)$ (differentiation).

**Solution:**
$T(a + bx + cx^2) = b + 2cx$

$\ker(T) = \{a + bx + cx^2 : b + 2cx = 0 \text{ for all } x\}$ Ôćĺ $b = 0, c = 0$

$\ker(T) = \{a : a \in \mathbb{R}\} = \text{span}\{1\}$, nullity = **1**.

Range$(T) = \{b + 2cx : b, c \in \mathbb{R}\} = P_1$, rank = **2**.

Check: $1 + 2 = 3 = \dim(P_2)$ Ôťô

---

**Q4.** $T: \mathbb{R}^2 \to \mathbb{R}^2$ is defined by $T(1, 0) = (2, 3)$ and $T(0, 1) = (1, -1)$. Find $T(4, 5)$.

**Solution:**
$T(4, 5) = 4 \cdot T(1, 0) + 5 \cdot T(0, 1) = 4(2, 3) + 5(1, -1) = (8, 12) + (5, -5) = (13, 7)$

## Key Formulas to Remember

| Formula | Description |
|---------|-------------|
| $\text{Rank}(T) + \text{Nullity}(T) = \dim(V)$ | Rank-Nullity Theorem |
| $\ker(T) = \{v : T(v) = \mathbf{0}\}$ | Kernel definition |
| $T$ injective $\iff \ker(T) = \{\mathbf{0}\}$ | One-to-one test |
| $T$ surjective $\iff \text{Range}(T) = W$ | Onto test |
| $A = [T(e_1) \mid T(e_2) \mid \cdots \mid T(e_n)]$ | Matrix of transformation |
